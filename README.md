# 📊 Retail Sales Forecasting Dashboard

## 🚀 Project Overview

This project analyzes Walmart retail sales data to uncover trends, detect anomalies, and forecast future sales using Python and Power BI.

---

## 📂 Project Structure

* 📁 Dashboard → Power BI dashboard (.pbix)
* 📁 Data → Dataset files
* 📁 Notebooks → Python analysis & forecasting
* 📁 SQL → SQL queries

---

## 🛠 Tools Used

* Power BI
* Python (Pandas, Matplotlib)
* Forecasting Models (Rolling Mean / Prophet)
* SQL

---

## 📈 Key Features

* Sales trend analysis
* Forecasting (Actual vs Predicted)
* Anomaly detection
* Store performance analysis
* Clean UI dashboard design

---

## 🎨 Dashboard Design

* Gradient UI (Figma-inspired)
* Card-based layout
* Hero section for key insights

---

## 📊 Insights

* Sales peaks during seasonal periods
* Larger stores generate higher revenue
* Forecast captures long-term trend effectively
